# examples

Example applications that demonstrate swift-rust features and serve as living documentation.

| Example         | What it shows                                            |
| --------------- | -------------------------------------------------------- |
| `hello-world`   | Smallest possible app, no special components.            |
| `with-image`    | Remote-source `<Image>` with allowlisted domains.        |
| `with-pdf`      | Server-side PDF generation via the Pdf component.        |

## Running any example

```bash
cd examples/hello-world
bun install
bun swift-rust dev
```
